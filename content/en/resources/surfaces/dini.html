In geometry, Dini's surface is a surface with constant negative curvature that can be created by twisting a pseudosphere. It is named after Ulisse Dini.


Dini's surface


Equations

Dini's surface described by the following parametric equations:


\begin{align} \ x = a\cos(u)\sin(v) \\ \end{align} \begin{align} \ y = a\sin(u)\sin(v) \\ \end{align} \begin{align} \ z = a\big( \cos(v) + \ln(\tan(\frac{v}{2})) \big) + bu \\ \end{align}



Maple Sample Code 


with(plots):
x := cos(u)*sin(v); y := sin(u)*sin(v);
z := cos(v)+log(tan((1/2)*v))+.2*u;
plot3d([x, y, z], u = 0 .. 4*Pi, v = .1 .. (1/2)*Pi, grid = [50, 20], title = `Dini's surface`);
x := x; y := y; z := z
