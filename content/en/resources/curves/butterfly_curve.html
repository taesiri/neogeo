The butterfly curve is a transcendental plane curve discovered by Temple H. Fey.

The butterfly curve.

Equations

The curve is given by the following parametric equations:

\begin{align} \ x = \sin(t) \Bigg( e^{\cos(t)} - 2\cos(4t) - \sin^5 \bigg( \frac{t}{12} \bigg) \Bigg) \\ \ y = \cos(t) \Bigg( e^{\cos(t)} - 2\cos(4t) - \sin^5 \bigg( \frac{t}{12} \bigg) \Bigg) \\ \end{align}

Polar equation:

\begin{align} \ r = e^{\cos(\theta)} - 2\cos(4\theta) + \sin^5\Bigg( \frac{2\theta - \pi}{24} \Bigg) \end{align}


Maple Sample Code 


with(plots);
R:=exp(cos(phi))-2*cos(4*phi) + sin(phi/12)^3;
plot([R,phi,phi=0..13*Pi],-4..4,-5..5,coords=polar);
